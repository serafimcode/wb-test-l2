Что выведет программа? Объяснить вывод программы. Объяснить внутреннее устройство интерфейсов и их отличие от пустых интерфейсов.

package main

import (
    "fmt"
    "os"
)

func Foo() error {
    var err *os.PathError = nil
    return err
}

func main() {
    err := Foo()
    fmt.Println(err)
    fmt.Println(err == nil)
}

Интерфейс является структурой с 2мя указателями, один из которых указывает на реальное зн-е присвоенное интерфейсу, а другой на тип этого зна-я.
Т.е. если интерфейсному типу присвоить какое то зн-е, то интерфейс уже не будет равен nil, даже если присвоенное зн-е само равно nil
