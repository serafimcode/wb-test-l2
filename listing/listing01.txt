Что выведет программа? Объяснить вывод программы.

package main

import (
    "fmt"
)

func main() {
    a := [4]int{76, 77, 78, 79, 80}
    var b []int = a[0:4]
    fmt.Println(b)
}

Вывод: 76, 78, 79. Так работает слайс через синтаксис [n,m], где n- индекс начала(включительно), m- индекс конца(не включительно)
